# java-cli-graalvm-hello-world

## Description
A POC for graalvm using native-image to build a standalone executable binary.

## Tech stack
- java
- native-image

## Docker stack
- ivonet/graalvm

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
